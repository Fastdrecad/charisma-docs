## 5. Measurement & Retention

### Key Metrics

**Engagement Metrics:**

- **Weekly Active Users (WAU):** Target 30%+ (dating apps have higher engagement)
- **Daily Active Users (DAU):** Target 15%+
- **Magic Openers Generated:** Target 3+ per active user/week
- **Sessions Created:** Target 20%+ users try Sessions feature (if Phase 4 complete)
- **Bookmark Rate:** Target 50%+ users bookmark at least 1 opener

**Retention Metrics:**

- **Day 1 Retention:** Target 60%+ (come back after first use)
- **Day 7 Retention:** Target 40%+ (weekly active)
- **Day 30 Retention:** Target 18%+ (monthly active)

**Product-Market Fit Signals:**

- **Net Promoter Score (NPS):** Target 50+ ("Would you recommend to a friend?")
- **App Store Rating:** Target 4.5+ (from beta testers)
- **Organic Word-of-Mouth:** Users sharing with friends
- **"When is full launch?"** questions (strong signal)

**Technical Metrics:**

- **Crash Rate:** Target <2%
- **Magic Openers Latency:** Target <5s average
- **OCR Processing Time:** Target <30s average (if Sessions Phase 4 complete)
- **API Uptime:** Target 99%+

### Analytics Setup

**Tools:**

- **PostHog** or **Mixpanel** (product analytics)
- **Amplitude** (user behavior)
- **Sentry** (error tracking)
- **Custom dashboard** (Supabase/PostgreSQL)

**Events to Track:**

- `app_opened`
- `magic_opener_generated`
- `magic_opener_bookmarked`
- `session_created` (if Phase 4 complete)
- `ocr_uploaded` (if Phase 4 complete)
- `ai_suggestion_generated` (if Phase 4 complete)
- `onboarding_completed`
- `feedback_submitted`
- `credit_purchased` (if RevenueCat integrated)

**Dashboards:**

- Daily active users
- Magic Openers generated per day
- Sessions created per day
- Credit consumption rate
- Onboarding funnel drop-off
- Error rate by feature

### Weekly Check-ins

**Monday:**

- Review analytics dashboard
- Send feedback survey email

**Wednesday:**

- Review feedback responses
- Prioritize bug fixes
- Plan feature improvements

**Friday:**

- Release bug fixes
- Deploy feature improvements
- Send "Week in Review" email to beta testers
